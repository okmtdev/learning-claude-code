argument-hint: (引数なし)

これはサブエージェントの動作を可視化するデモです。
別ターミナルで `./scripts/watch-agents.sh` が起動している前提で、以下を **1つのメッセージ内で並列に** 実行してください（依存関係はありません）:

1. `code-explorer` サブエージェントに「sandbox/src 配下にある関数の一覧と、それぞれの役割を1行で説明して」と依頼する。
2. `test-runner` サブエージェントに「sandbox のテストを実行して結果を報告して」と依頼する。
3. `code-explorer` サブエージェントに「このリポジトリで定義されているカスタムスラッシュコマンドを全て挙げて」と依頼する。

3つのサブエージェントが起動・完了する様子が Monitor に 🚀 SPAWN / ✅ DONE として流れるはずです。
全エージェントの結果が返ってきたら、それぞれの要約を箇条書きでまとめてください。
