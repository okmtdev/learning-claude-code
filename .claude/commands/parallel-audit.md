これはマルチエージェント・オーケストレーションのデモです。
別ターミナルで `./scripts/watch-agents.sh` を起動しておくと、多数の 🚀 SPAWN / ✅ DONE が流れます。

次の調査を **1メッセージ内で並列に** 実行してください（互いに依存しません）:

1. `code-explorer` に「sandbox/src 配下の各関数の役割を一覧化」させる。
2. `code-explorer` に「このリポジトリのカスタムコマンドとサブエージェントを列挙し、用途を要約」させる。
3. `reviewer` に「sandbox/src/textUtils.js と mathUtils.js をレビューし、改善余地を指摘」させる。
4. `test-runner` に「sandbox のテストを実行して結果を報告」させる。

4つの結果が出揃ったら、**メインエージェントが統合役**として:
- 各エージェントの要点を1つの表/箇条書きにまとめる。
- 矛盾や重複があれば指摘する。
- 「次にやるべきこと」を優先度つきで3つ提案する。

ポイント: 調査(読み取り)を並列で分担し、最後に1人が統合する「fan-out → 統合」パターンです。
