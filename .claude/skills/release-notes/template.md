# リリースノート出力テンプレート

このフォーマットに沿って出力すること（[Keep a Changelog](https://keepachangelog.com/) 風）。

```markdown
## [バージョン or 日付]

### Added（追加）
- ...

### Changed（変更）
- ...

### Fixed（修正）
- ...

### その他
- ...
```

注意:
- 該当するコミットが無いセクションは省略してよい。
- 各項目は利用者目線で1行に要約（コミットメッセージそのままでなくてよい）。
- 末尾に「対象範囲（例: v1.0.0..HEAD）」を一言添える。
